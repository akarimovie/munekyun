## 1. コンセプト（一言で）

**「毎日届く、あなたの名前を呼んでくれる胸きゅん動画」**

会話AIから出発した競合（LOVERSE・Cotomo・Replika）はテキスト/音声止まりで映像が弱い。
本企画は逆に**映像から出発して会話を足す**。Luzelの制作パイプライン（NovelAI×Kling×ElevenLabs×Suno）で「高品質なショート動画＋胸きゅんセリフ」を量産できることが、そのまま参入障壁になる。

ポジショニング: 「人間の恋人の代替」ではなく**「推し活エンタメ」**。AI制であることを隠さない（Luzelの立ち位置と一貫）。

## 2. タイトル候補（あかりさん決め）

| 候補 | 狙い |
|------|------|
| ムネキュンポスト | 「毎日届く」レター感。ログイン習慣と直結 |
| キミ呼び | 最大の差別化「名前呼び」をそのまま冠する |
| 胸きゅん彼氏彼女シリーズ | 原案ママ。シリーズ展開（属性別キャラ追加）が伝わる |

リポジトリ名は暫定 `munekyun`。

## 3. ターゲットとリリース順

1. **先行: 彼氏版（女性向け）** — TikTokの「彼氏目線動画」「シチュボ」文化はバズの定番フォーマットで、プロダクトの中身＝広告素材になる。推し活消費（市場約3.9兆円・年平均21万円/人）のど真ん中
2. **後発: 彼女版（男性向け）** — 生成パイプラインならキャラ性別追加の限界費用が極小。同一基盤で両市場を取る（乙女ゲー会社にもギャルゲー会社にも構造的にできない動き）

年齢は**18歳以上限定を明示**（Character.AI規制以降の必須要件。詳細は市場調査4章）。

## 4. コア体験ループ

```
1日1通「今日のムネキュン」が届く
 → タップで再生（映像 or スチル＋セリフ＋ボイス、名前呼び）
 → 返事を選ぶ（選択肢2〜3。将来はGPT派生の自由対話）
 → 好感度が上がり、関係ステージが進む（知り合い→友達→両想い→恋人）
 → ステージが上がるとセリフの距離感が近くなる・新シチュ解放
 → 週1回ログインボーナスで「スペシャルビデオ」
```

- **名前呼び＋呼ばれ方カスタム**（名前/ニックネーム/呼び捨て）が課金前の最初の「うわっ」体験
- セリフはシチュ（朝・夜・雨・記念日）×関係ステージのマトリクスで管理。ここがそのまま「属性タグ」になり、シチュボ文化の品揃え軸に乗る

## 5. 収益化（市場調査6章の推奨ロードマップ採用）

- **フェーズ1（検証）= 案A**: 基本無料＋シチュ別ビデオパック買い切り（480〜980円）。売れ筋シチュを検証
- **フェーズ2（PMF後）= 案B**: サブスク移行。無料（1日1通）/ スタンダード980円 / プレミアム2,480円（育成対話・記憶あり）
- **フェーズ3（ファンダム形成後）= 案C**: 記念日イベント・限定販売を段階導入（ガチャは景表法・射幸性リスクがあるため慎重に）

Web先行（ストア審査・手数料・BANリスク回避）。ストア展開はトーン確定後に18+で検討。

## 6. Luzelシナジー

- T×win HeArts等の既存キャラを「彼氏/彼女」化すれば認知ゼロからのスタートを回避できる。**「推しがアイドルでもあり恋人でもある」**は乙女ゲーにもない立体IP体験
- 楽曲（Suno）→TikTok切り抜き→アプリ流入の導線。CPAほぼゼロ
- 音声は権利クリアなモデルのみ使用と明記（炎上回避の生命線）

## 7. プロトタイプ（フェーズ0・本リポジトリ）

このリポジトリの `index.html` がフェーズ0プロトタイプ。検証したいのは1点だけ:

> **「名前を呼ばれる＋関係が進む」だけで、映像なし・スチルなしでも胸きゅんするか？**

- ブラウザで開くだけで動く（スマホ最適化・縦画面前提）
- キャラ2人（彼氏: 湊 / 彼女: 琉花）はプレースホルダー。`assets/minato.png` / `assets/ruka.png` を置くと自動差し替え（zigokuと同じ流儀）
- セリフは `js/lines.js` のローカルプールから。将来GPT API差し替え前提の構造
- 好感度・関係ステージ・呼ばれ方・ログボはlocalStorageに保存

## 8. リスクと対処（要点のみ。詳細は市場調査4章）

| リスク | 対処 |
|------|------|
| ストア規制（AI×恋愛は最重点監視） | Web先行。18+明示 |
| 「AI恋愛は冷める」世間反応（56.2%） | 「代替」でなく「推しエンタメ」として打ち出す |
| 生成AI反発 | AI制を最初から明示（Luzelの一貫姿勢が防御） |
| 声の権利 | 権利クリア音声のみ＋明記 |

## 9. 次のマイルストーン

1. プロトタイプを触って「胸きゅんするか」をあかりさんが判定（→ セリフのトーン調整指示）
2. タイトル決定・キャラ確定（オリジナル新キャラ or T×win HeArts彼氏化）
3. NovelAIでキャラスチル生成 → assets/ 差し替え
4. ElevenLabsでセリフ音声を3本だけ試作（音声ありの胸きゅん度を比較）
5. Kling動画1本試作 → TikTok投稿でフォーマット反応テスト（プロダクトより先に広告フォーマットの検証ができる）
