# 胸きゅん彼氏彼女シリーズ（プロトタイプ）

Notionゲーム制作案 #9。**「毎日届く、あなたの名前を呼んでくれる胸きゅん動画」**のフェーズ0プロトタイプ。

- 企画の全体像 → [docs/企画書.md](docs/企画書.md)
- 市場・競合・収益化の調査 → [docs/市場調査_競合分析.md](docs/市場調査_競合分析.md)

## 遊び方

```
open index.html
```

ブラウザで開くだけ。スマホ縦画面に最適化（PCでも動く）。

1. キャラを選ぶ（彼氏: 湊 / 彼女: 琉花）
2. 呼んでほしい名前と呼ばれ方（呼び捨て/ちゃん/さん等）を設定
3. 「💌 今日のムネキュン」でセリフが届く → 返事を選ぶと好感度UP
4. 関係ステージ: 知り合い → 友達 → 両想い？ → 恋人（ステージでセリフの距離感が変わる）
5. 7通ごとにスペシャル演出（製品版は週1ログボ想定）

本番は1日1通だが、検証用に「もう1通（プロト検証モード）」ボタンで連続再生できる。
進行はlocalStorage保存。「最初からやり直す」でリセット。

## このプロトタイプで検証したいこと

> 名前を呼ばれる＋関係が進む、それだけで（映像なしでも）胸きゅんするか？

## キャラ絵の差し替え

`assets/minato.png` / `assets/ruka.png` を置くと自動でアバターに反映される（正方形・顔中心推奨）。

## 構成

```
index.html      … エントリ
css/style.css   … 画面スタイル（キャラ別テーマ）
js/lines.js     … セリフデータ（ステージ×時間帯。将来GPT API差し替え前提の構造）
js/app.js       … 本体（画面遷移・好感度・演出・セーブ）
docs/           … 企画書・市場調査
assets/         … キャラ画像置き場
```
